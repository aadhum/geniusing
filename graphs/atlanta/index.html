This network visualizes the relationships between a selected number of hip hop artists from Atlanta and their producers.

  • Ludacris
  • T.I.
  • Gucci Mane
  • Migos
  • OutKast

The wider edges represent more collaborations between the artist and a producer. The nodes that are larger indicate that the producer worked with more of the artists

The data represents 2,829 songs that were collected from the Genius API using this Jupyter Notebook.

If you want to play with the graph interactively try downloading the Cytoscape session file or using the edge-list CSV in your favorite network visualization environment.
